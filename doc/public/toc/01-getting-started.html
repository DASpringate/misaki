Getting Started

    Running sample

    $ git clone git://github.com/liquidz/misaki.git
$ cd misaki
$ lein run sample

    Access http://localhost:8080

    Make your own blog

    Copy sample directory.

    $ cp -pir sample your_blog

    Run develop server.

    $ lein run your_blog

    Edit templates. If you update template files, misaki will compile them automatically, and you can preview at http://localhost:8080immediately.

    $ vi your_blog/_templates/index.html.clj
